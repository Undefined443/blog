## [tailspin](https://github.com/bensadeh/tailspin)

安装：

```sh
# macOS
brew install tailspin

# Cargo
cargo install tailspin
```

使用：

```sh
tspin <file>
```

## [lnav](https://lnav.org/)

安装：

```sh
# macOS
brew install lnav

# Ubuntu
sudo snap install lnav
```

使用：

```sh
lnav <file>
```
